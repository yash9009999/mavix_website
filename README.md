# MavixHub - Premium Futuristic 3D Website

A cutting-edge, premium 3D website for MavixHub - a marketing agency, content writing service, and IT consultancy company built with Next.js and modern web technologies.

## 🚀 Features

- **Futuristic 3D Design** with Three.js and React Three Fiber
- **Glassmorphism UI** with blur effects and transparency
- **Smooth Animations** powered by Framer Motion and GSAP
- **Interactive 3D Experience** with floating elements and particle systems
- **Responsive Design** optimized for all devices
- **Premium Lead Capture** form with validation
- **Google Maps Integration** for office location
- **Custom Cursor** with hover effects
- **SEO Optimized** structure and metadata

## 🛠️ Tech Stack

- **Framework**: Next.js 14 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion, GSAP
- **3D Graphics**: Three.js, React Three Fiber, React Three Drei
- **Icons**: Lucide React
- **Fonts**: Poppins, Inter, Orbitron

## 📦 Installation

1. **Clone the repository**
   ```bash
   git clone <repository-url>
   cd mavixhub
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Run the development server**
   ```bash
   npm run dev
   ```

4. **Open your browser**
   Navigate to [http://localhost:3000](http://localhost:3000)
